Hello! and welcome to my Aerial Image Explorer Tool!

This interface is a leisure project of mine spinoff from an extensive research on using open access Satellite imagery and Computer Vision to extract urban and built-up features from cities and large scale infrastructure to aid in their urban planning. Read more about the research here. Aerial Image Explorer v2 aims to package the Python based imagery retrieval workflow into a full-fledged app where anyone can browse, explore and collect high-res image from Sentinel. This app is developed in Flask and Jinja2 using a Pythonic Backend. Do note satellite image API requests often take 30+ seconds to return and are limited by HTTP request response times. Try with lesser number of images to get the best output as this project is still in development.

For simplicity, I've decided to have a few methods by which one can explore these beautiful pieces of data.



  • By Airport: Get Image of an Airport by Name

  • By Country: Get Images of all airports in a Country

  • By Coordinates: Get Image of a custom Latitude, Longitude pair

  • By Traffic: Get Images Airports by order of their Passenger Traffic
